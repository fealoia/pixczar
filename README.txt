How to Compile: "ocamlbuild pixczar.native"

How to Execute: "./pixczar.native [file.pxr]"

How to Run Test Script: "./tester.sh" 
 - Green text indicates expected output for a test
 - Red text indicates unexpected output for a test
    -Ex: "./pixczar.native shouldfail.pxr" should print FAILURE in green

Syntax to be Added: As of now, we do not plan to add any new syntax
